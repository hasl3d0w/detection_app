🎥 Live Object Detection, Counting & Alerts (Streamlit + YOLOv8)

This project is a real-time object detection web application built using **Streamlit**, **YOLOv8 (Ultralytics)**, and **streamlit-webrtc**. It detects objects from your webcam, counts them, and displays alerts when a selected object exceeds a specified threshold.

---

🚀 Features

- 📷 Real-time webcam object detection
- 🎯 Select specific objects to track (e.g., person, car, etc.)
- 🔢 Live object counting
- 🚨 Alert system when object count exceeds a threshold
- ⚡ Fast processing using YOLOv8

---
 🛠️ Technologies Used

- Python
- Streamlit
- streamlit-webrtc
- Ultralytics YOLOv8
- OpenCV
- PyAV
- GitHub
